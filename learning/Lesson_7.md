## 6. –ú–∏–Ω–∏–º–∞–ª—å–Ω—ã–π –∫–æ–Ω–≤–µ–π–µ—Ä –ø–æ–¥–≥–æ—Ç–æ–≤–∫–∏ —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏–∏ (–±–µ–∑ –æ—Ç–ø—Ä–∞–≤–∫–∏)

–í —ç—Ç–æ–º –ø—Ä–∏–º–µ—Ä–µ –º—ã **–Ω–∏—á–µ–≥–æ –Ω–µ –æ—Ç–ø—Ä–∞–≤–ª—è–µ–º**, —Ç–æ–ª—å–∫–æ —Å–æ–±–∏—Ä–∞–µ–º —Å—Ç—Ä—É–∫—Ç—É—Ä—É ‚Äî —á—Ç–æ–±—ã —É–≤–∏–¥–µ—Ç—å, –∫–∞–∫ `solana` –∏ `solders` —Ä–∞–±–æ—Ç–∞—é—Ç –≤–º–µ—Å—Ç–µ.

```python

from solders.pubkey import Pubkey

recent_blockhash = client.get_latest_blockhash().value.blockhash

message = Message.new_with_blockhash([ix], sender.pubkey(), recent_blockhash)

tx = Transaction([sender], message, recent_blockhash)
tx.partial_sign([sender], recent_blockhash)

raw = bytes(tx)
print("Tx bytes:", len(raw), "–±–∞–π—Ç")
```

> ‚ö†Ô∏è **–ü—Ä–µ–¥—É–ø—Ä–µ–∂–¥–µ–Ω–∏–µ:** —É –Ω–æ–≤–æ–≥–æ –∫–æ—à–µ–ª—å–∫–∞ **–Ω–µ—Ç –±–∞–ª–∞–Ω—Å–∞**, –ø–æ—ç—Ç–æ–º—É —Ä–µ–∞–ª—å–Ω–∞—è –æ—Ç–ø—Ä–∞–≤–∫–∞ –∑–∞–≤–µ—Ä—à–∏—Ç—Å—è –æ—à–∏–±–∫–æ–π *AccountNotFound*. –ó–¥–µ—Å—å –º—ã —Ç–æ–ª—å–∫–æ –ø–æ–∫–∞–∑—ã–≤–∞–µ–º **—Å—Ç—Ä—É–∫—Ç—É—Ä—É**.
